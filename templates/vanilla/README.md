# SMNTC Vanilla Starter

A minimal Three.js + SMNTC starter powered by Vite.

## Quick Start

```bash
npm install
npm run dev
```

Open the URL printed in your terminal to see the surface.

## Customize

Edit `main.js` to change tokens:

```js
const kernel = new SMNTCKernel({
  surface: 'topographic',
  vibe: 'calm',
  palette: 'monochrome',
});
```
